1. Telécharger Mozilla Firefox 54.0 et le plug in Selenium IDE pour Mozilla Firefox
2. Télécharger le plug in Flow control pour Selenium : https://addons.mozilla.org/fr/firefox/addon/flow-control/
3. Ouvrir Selenium > Options > Options ... > Extensions de Selenium IDE : rechercher le fichier user-extensions.js
4. Redémarrer Selenium pour appliquer
